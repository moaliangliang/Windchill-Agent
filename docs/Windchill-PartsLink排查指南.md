# Windchill PartsLink / 分类排查指南

## 概述
- **用途**: PartsLink 分类管理和配置问题排查

## 1. 分类节点不显示
- 检查分类节点是否已发布
- 确认用户有分类访问权限
- 检查分类配置文件

## 2. 分类属性加载失败
- 检查节点->属性的映射关系
- 确认分类 XSD 文件正确

## 3. PartsLink 搜索无结果
- 检查分类索引是否正确构建
- 确认节点分配正确
